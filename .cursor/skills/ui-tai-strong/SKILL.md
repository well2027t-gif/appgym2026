# UI TAI Strong

## Objetivo

Transformar referências visuais em telas reais com aparência premium, clara e consistente.

## Instruções

1. Priorize mobile first.
2. Preserve a identidade do app e o fluxo já existente.
3. Reaproveite padrões já usados no projeto antes de criar novos.
4. Ao receber um print, ajuste primeiro:
   - hierarquia visual
   - espaçamento
   - tamanhos
   - bordas
   - sombras
   - cores
5. Prefira mudanças pequenas e visíveis em vez de refazer a tela sem necessidade.
6. Se a tela estiver grande demais para o viewport, reduza alturas, paddings e gaps antes de remover conteúdo.

## Checklist rápido

- A tela cabe bem em celular?
- Os botões principais estão destacados?
- O visual combina com `Home`, `Perfil` e `Treino`?
- O print de referência foi respeitado?
